# Quick Start Guide - Phase 2 Project

## 🚀 Get Started in 3 Steps

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Choose Your Mode

#### Option A: With TGAM1 Hardware
```bash
python main.py
```
- Connect TGAM1 via Bluetooth
- Enter COM port in GUI
- Click "Connect" → "Start Monitoring"

#### Option B: Demo Mode (No Hardware)
```bash
python demo_mode.py
```
- Simulates EEG data
- Tests all components
- Perfect for development/testing

### Step 3: Monitor & Analyze
- Watch real-time visualization
- Observe fatigue detection
- Receive alerts when thresholds exceeded
